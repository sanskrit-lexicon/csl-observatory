## How to extend this digest

Future quarterly digests live at `docs/trends/YYYY-QN.md`. The tracking workflow is:

1. arXiv RSS feeds for `cs.DL` (Digital Libraries) and `cs.CL` (Computation and Language)
2. Google Scholar alerts (manual setup by user)
3. Semantic Scholar API queries for keywords: `digital humanities`, `digital lexicography`, `computational stemmatics`, `TEI`, `OntoLex`, `Sanskrit lexicography`, `Cologne digital`
4. Manual scan of conference websites (WSC, ELEX, DH, DH-Russia, ICOSAL)
5. Wiktionary + Wikidata API for monthly statistics updates

Synthesis happens at end of each quarter (March 31, June 30, September 30, December 31) and is committed as a new file in this directory.
