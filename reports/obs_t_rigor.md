# Statistical rigor — two-axis typology (OBS-T)

_Generated by `scripts/obs_t_rigor.py`. The typology is two orthogonal axes: **location** (where in the entry, derived labels) and **edit-type** (what kind of change, from edit-ops). Tested with Wilson CIs, chi-square (Cramér's V), and Mann-Kendall trends._

## H1 — corrections are micro surface edits, at every location

Edit distance: **median 2**, **66% are ≤ 2 characters** (p90 18, max 404). Edit-type shares:

| edit type | share |
|---|---:|
| spelling | 33.3% |
| punctuation | 20.0% |
| spacing | 19.1% |
| diacritic | 10.6% |
| case | 8.7% |
| digit | 5.2% |
| none | 1.7% |
| transposition | 1.4% |

Every type is a surface change — there is no "content rewrite" category. Crucially this holds **across locations**: the minor-edit rate (a small surface edit) is high even where meaning lives.

| location | n | minor-edit rate (95% CI) |
|---|---:|---|
| sense | 17,880 | 69.8% [69.1%, 70.5%] |
| headword | 7,282 | 81.5% [80.6%, 82.4%] |
| markup | 3,945 | 18.2% [17.1%, 19.5%] |
| citation | 3,338 | 30.7% [29.2%, 32.3%] |
| meta | 822 | 88.6% [86.2%, 90.6%] |
| grammar | 294 | 31.6% [26.6%, 37.2%] |

So even corrections **located in the sense/definition and headword** are overwhelmingly small form fixes, not redefinitions — the honest, two-axis restatement of the old "surface dominates" claim.

## H2 — the LOCATION profile differs by dictionary (derived labels)

Chi-square independence of location × dictionary (top 15, derived only): χ² = 23674.5, dof = 70, p < 0.001, **Cramér's V = 0.415**. Dictionaries differ in *where* their errors sit, not only how many — now on clean location labels.
